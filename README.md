## 🚀 Quick Start

### 1. Build & Deploy
```bash
npm install
npm run build
```

The `dist` folder contains your production-ready website.

### 2. Admin Access
- URL: `https://yourdomain.com/admin`
- Credentials: See `ADMIN_CREDENTIALS.md`

### 3. Update Configuration
Before deploying, update:
- `src/config/seo.ts` - Add your real contact info
- `index.html` - Replace domain references
- `public/sitemap.xml` - Replace domain references
- `public/robots.txt` - Replace domain references
