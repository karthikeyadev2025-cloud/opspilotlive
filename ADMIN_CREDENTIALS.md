# Admin Credentials

## IMPORTANT: First Time Setup Required

Before you can access the admin panel, you MUST create an admin account. This is a one-time setup using Supabase authentication.

## How to Create Your Admin Account

### Method 1: Using the Admin Login Page (Recommended)

1. Visit your website
2. Scroll to the footer and click the "Admin" link
3. On the login page, click "First time? Create Admin Account"
4. Enter your email: **admin@aadyaenterprises.com** (or use your own)
5. Create a strong password (minimum 6 characters)
6. Click "Create Account"
7. You'll be automatically signed in

### Method 2: Using Supabase Dashboard

1. Go to https://supabase.com and log into your project
2. Navigate to Authentication > Users
3. Click "Add User"
4. Enter:
   - **Email:** admin@aadyaenterprises.com
   - **Password:** Your secure password
5. **IMPORTANT:** Enable "Auto Confirm User"
6. Click "Create User"
7. Now you can log in at `/admin`

## How to Access Admin Panel After Setup

1. Visit your website
2. Scroll to the footer
3. Click on the "Admin" link
4. Enter your email and password
5. Click "Sign In"

## Security Notes

- Change the default password immediately
- Use a strong, unique password
- Never share admin credentials
- Enable 2FA if available in your Supabase project
- Regularly update passwords

## Admin Features

Once logged in, you can manage:

- Site Content (Hero, Contact info, etc.)
- Managing Director Information & Photo
- Services (Add/Edit/Delete)
- Solar & CCTV Product Details
- Team Technicians
- Benefits Section
- Gallery Images
- Customer Testimonials
- Career Applications (View/Manage)
- Investment Inquiries (View/Manage)
- Contact Form Settings

## Forgot Password?

Use Supabase's password reset functionality:
1. Go to the admin login page
2. Request password reset
3. Check your email for reset link
4. Or reset directly from Supabase Dashboard
