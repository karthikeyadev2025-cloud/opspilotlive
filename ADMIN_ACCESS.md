# Admin Panel Access

The admin panel is accessible at: **https://aadyasolars.com/admin**

## Admin Credentials

**Email:** `admin@aadyaenterprises.com`
**Password:** `Admin@2024`

## Features

The admin panel allows you to manage all website content without any database:

- Site Content (Hero, About, Contact)
- MD Information
- Services
- Solar & CCTV Details
- Technician Team
- Benefits
- Gallery Images
- Testimonials
- Career Applications
- Investment Inquiries
- Settings

All data is stored locally in the browser's localStorage and persists across sessions.

## Important Notes

- No database or backend required
- All changes are saved instantly
- Data is stored in browser localStorage
- Clear browser data will reset all content
- Admin session persists until logout
